Nginx 新手入门

Jun 22, 2016

启动

启动 nginx 可以用
• sudo nginx
• sudo service nginx start，service 命令会从 /etc/init.d 目录中找到对应的脚本（这里是 nginx）并执行

一定要用 root 权限启动，否则 log 文件没有写权限，导致启动失败

热配置

nginx -s signal，其中 signal 可以是下面中的一个：
• stop，快速停止
• quit，等待请求处理完成后再停止
• reload，重新加载配置文件（这个很有用）
• reopen，重新打开日志文件

配置文件

配置文件的指令分为简单指令和块指令
• 简单指令，用空格分隔的键值对，分号结尾
• 块指令，花括号包住的一系列指令组成

  • http 对应的块也叫做上下文，不在任何指令块中的指令，它的上下文是 main
  • server 匹配的是 url 域名和端口部分
  • location 匹配的是 url 路径部分，匹配方式可以是前缀匹配、正则表达式匹配等
